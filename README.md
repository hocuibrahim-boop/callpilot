# CallPilot — Laravel Backend

Emlak çağrı yakalama ve müşteri takip sistemi.

## Kurulum

GitHub Actions ile otomatik deploy edilir. Sunucuda composer gereklidir.

## API Endpoints

- POST /api/auth/login
- POST /api/devices/register  
- GET/POST /api/calls
- GET/POST /api/leads
- GET/POST /api/reminders
- GET /api/panel/ozet (yönetici)
- POST /webhook/verimor

## Gereksinimler

- PHP 8.2+
- MySQL 8+
- Composer
